Basic Structure of HTML Page:

<!DOCTYPE html>
<html>
 <head>
  <title>Title Here</title>
 </head>
 <body>
  <h2>HTML Structure</h2>
  <p>Tell something about the Web page here.</p>
  <nav>
   <ul>
    <li><a href="#section1">Section 1</a></li>
    <li><a href="#section2">Section 2</a></li>
    <li><a href="#section3">Section 3</a></li>
   </ul>
  </nav>
  <section id="section1">
   <h2>Section 1</h2>
    <p>This is the content of section 1.</p>
  </section>
   <footer>
    Copyright © 2023 Web page
   </footer>
 </body>
</html>

This is how your code should look for a webpage. Adding different HTML elements will make your page stand out and interest the user more. If you indent your code properly it will be easier to read and debug if you come across any errors. Also, indenting can help to section your code off which will make testing and again, debugging, much faster.
