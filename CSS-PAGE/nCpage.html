CSS code will usually look like this:

.classname1 {
    color: white;
    font-size: 16px;
    text-align: center;
    position: relative;
    etc....

If you have previous experience with JavaScript programming, you will know that this closely resembles the structure of 'Objects' in the language.
For this example, a class would be created This is why there is a dot (.) before the classname1.

It would look like this in your code:

<div id="idName">Some text</div>

Ways to include CSS styling:

One way is to insert it within the HTML document in the 'Head' section. Although, this is considered bad practice and you should instead include it this way.

The most common way is to create an external CSS file and insert it via linking this file in the 'Head' section on your HTML document.

#classname2 {
    color: black;
    font-weight: bold;
    padding: 20px 0 20px 0;
    background-color: #333 ;
    align-items: center ;
    etc....
When you create a <div id="idName">Some text</div>, you will need to use a hashtag (#) before you put the 'idName', curly brackets, and then your styling code
As seen above.

There are hundreds of CSS styling elements. You won't be able to memorise them all but knowing the basic ones will help start you off.

  • color (changes the text color)
  • background-color (changes the background color)
  • font-size (changes the font size)
  • font-family (changes the style of font e.g Arial)
  • border (to create a visible border you must put: solid, 'color of choice', size in pixels)
  • margin (creates space around elements outside of any borders
  • padding (creates space around an elements content inside of any borders)
