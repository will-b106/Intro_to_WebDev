As you've heard before, JavaScript is used in web dev to add interaction for the user. For example, the search bar on this page uses JavaScript. When you click on the Search bar, a function is called which makes the search bar take over the full screen. Then another function is called when you click the Back button to set the search bar display to none (otherwise stopping the first function).

You can use the JavaScript course I have attached in the classroom to learn the JavaScript basics you will need for Web Dev. If you are already quite confident with JS then you can practice Js exam questions via the link on this page. Another good way to learn or practice JavaScript is to download an app called Mimo. You can download it on your phone and choose the JavaScript course and then either start right at the beginning or review certain aspects of JS you're not comfortable with.
